## 🎯 프로젝트 소개

**SotoLab 2026**은 22년 경력의 개발 멘토 장성균 박사의 멘토링 서비스를 소개하는 프리미엄 비즈니스 웹사이트입니다.

### 핵심 가치

- ✅ **22년 경력의 전문성**: 공학박사, 한양대 겸임교수 역임
- ✅ **500+ 멘티 배출**: 98% 만족도, 평균 6개월 내 취업/이직 성공
- ✅ **SOTE 학습법**: 체계적인 4단계 학습 방법론
- ✅ **실제 프로젝트 포트폴리오**: 9개 MVP 프로젝트 (IoT, AI, 블록체인, 모바일)
- ✅ **7권의 저서**: 인문학부터 전공서까지
- ✅ **온라인 강의**: 블록체인 이해와 구조 23강

## 📄 페이지 구성

### 1. **Home** (index.html)
- Hero 섹션 + 4개 성과 지표
- 성장 임팩트 3가지
- 6개 멘토링 서비스
- 소토 멘토 소개 (22년 경력)
- 2개 후기 슬라이더
- CTA 섹션

### 2. **MVP 프로젝트** (mvp-projects.html)
- 필터 탭 (전체/웹/모바일/AI/IoT/블록체인)
- 9개 프로젝트 카드 (YouTube 영상 임베드)
  - English Master (Mobile)
  - 보안 파일 지문인식 (IoT)
  - AIRecognition (AI)
  - 스마트미러 (IoT)
  - 에코브리티 (IoT)
  - Jeju 4.3 희생자 기록 (Blockchain)
  - 제주 감귤 나무 분양 (Blockchain)
  - 스마트 게스트 하우스 (IoT)
  - 삼도 코인 (Blockchain)

### 3. **커리큘럼** (curriculum.html)
- 교육철학: 소토피아(Sotopia)
- 멘토링 핵심 요소 4가지
- SOTE 학습법 4단계 타임라인
  - S: STUDY (전공서적 스터디)
  - O: NOTE (노트 정리)
  - T: TEACH (가르치듯 발표)
  - E: REVIEW (스터디 리뷰)
- 핵심 4가지 & 혜택 6가지

### 4. **출판도서** (books.html)
- 인문학: 멘토티 멘토링 (2022)
- 전공서 6권:
  - 블록체인 멘토링 (2019)
  - 블록체인 앱 메이커 (2019)
  - 블록체인 투표시스템 (2018)
  - 미니컴퓨터 메이커 (2018)
  - 임베디드 실무론 (2016)
  - 임베디드 스케치 (2011)

### 5. **온라인강의** (online.html)
- 블록체인 이해와 구조 [23강]
- 강의 경력 (한양대, 연세대, 제주대, KISA, 한컴MDS 등)
- 강의 신청 링크

### 6. **멘토링후기** (reviews.html)
- 8개 실제 후기 (상세 2개 + 간결 6개)
- 필터링 기능
- 성과 통계 (500+ 멘티, 98% 만족도, 22년 경력, 6개월 취업)

### 7. **멘토링문의** (contact.html)
- 상담 신청 폼 (7개 입력 필드)
- 연락처 정보 4가지
  - 카카오톡 오픈채팅
  - 이메일
  - GitHub
  - 블로그
- FAQ 5개 (아코디언)

## 💻 로컬 실행 방법

### 방법 1: Python
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

### 방법 2: Node.js
```bash
npx http-server -p 8000
```

### 방법 3: VS Code
- Live Server 확장 프로그램 설치
- `index.html` 우클릭 → "Open with Live Server"

브라우저에서 `http://localhost:8000` 접속
