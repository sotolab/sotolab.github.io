OOO 직거래 모바일 앱

1. 앱 개요

앱 이름: OOO 마켓 (가칭)

목적: 생산자와 구매자 간의 직거래 플랫폼 제공

중간 유통 비용을 줄이고, 신속하고 투명한 거래를 지원하는 것이 목표입니다.

2. 주요 사용자

  • 판매자: 생산자
  • 구매자: 종사자

3. 핵심 기능

  • OOO 등록 및 판매
  • 실시간 채팅 기능 (구매자-판매자 직접 소통)
  • 카카오페이를 통한 안전 결제
  • 위치 기반 거래 (카카오맵 연동)
  • 상품 검색 및 필터링 (지역, 가격, 품질 기준)

4. 적용 기술 스택

  • 프론트엔드: React Native (iOS & Android)
  • 백엔드: Django REST Framework (DRF)
  • 데이터베이스: MySQL
  • 결제 시스템: 카카오페이 API 연동
  • 지도 서비스: 카카오맵 API 연동
  • 실시간 채팅: Firebase Firestore
